# RoomMonitorESP8266
Room Monitoring (Air Quality, Temperature and Humidity) with ESP8266. the results are in 4 ways:
- OLED DIsplay
- WebServer
- JSON data
- Grafana (With PostgreSQL)

## Prerequites
- ESP8266
- Sensors (DHT22 & MQ-135)
- OLED Display (optional)
- Python 3.19+
- Grafana
- PostgreSQL/MySQL/any SQL framework
